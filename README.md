# Sentinel Risk Audit

**Framework de Evaluación de Resiliencia Operativa**

Herramienta diseñada para identificar vulnerabilidades críticas en la gestión de empresas de seguridad y servicios operativos. Este proyecto combina la experiencia en gestión de riesgos y seguridad ocupacional con desarrollo web funcional.

### ¿Por qué esta herramienta?
En entornos de seguridad física, el riesgo no es solo técnico, es operativo y legal. Una falla en la gestión del capital humano o el mantenimiento de equipo es una brecha abierta para cualquier negocio.

### Tecnologías
- JavaScript Nativo (ES6)
- CSS3 (Diseño modular y responsive)
- HTML5

### Uso
Esta herramienta permite a los gerentes de operaciones realizar un autodiagnóstico rápido de sus procesos críticos, entregando un nivel de riesgo basado en el cumplimiento normativo y operativo.

## Despliegue
Este proyecto está optimizado para ejecutarse directamente en cualquier navegador moderno mediante [GitHub Pages](https://pages.github.com/). No requiere compilación ni dependencias externas, asegurando máxima portabilidad y seguridad.

"La arquitectura del proyecto permite su uso en entornos offline (sin conexión a internet), ideal para auditorías de campo en sitios con acceso restringido."
